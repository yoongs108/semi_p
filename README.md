# 담당 페이지 설명 및 시연

## Zipper Project 에서 구현한 페이지
1. 마이 페이지
2. 클래스 동영상 CRUD
3. 결제한 클래스 리스트 페이지
4. 동영상 수강 페이지

## 마이 페이지
* 마이 페이지에서 정보수정, 결제한 내역, 작성한 게시글, 1:1 문의내역으로 연결됩니다.
* 세부 구현은 조원 별로 나누어 가져갔으므로 연결용 페이지 이며, session의 회원 정보를 이용하여 데이터를 불러옵니다.
* 개인이 이용하는 화면임을 메뉴에 닉네임을 불러 오는 것으로 나타내었습니다.
